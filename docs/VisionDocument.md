# Documento de Visão: Plataforma de Vendas e Doações de Roupas Usadas

## 1. Introdução
### Objetivo do Projeto
Desenvolver uma aplicação web para conectar doadores, vendedores e compradores de roupas usadas, promovendo a economia circular e reduzindo o desperdício têxtil e visando a sustentabilidade.

### Escopo do Produto
- Sistema de vendas e doações com cadastro de usuários
- Catálogo de produtos com filtros avançados
- Integração com gateways de pagamento
- Sistema de reputação e avaliações
- Painel administrativo para moderação

## 2. Posicionamento
### Declaração de Posição
*Para consumidores conscientes que buscam alternativas sustentáveis na moda, a ReVeste é uma plataforma digital que facilita a venda e doação de roupas usadas, conectando pessoas e reduzindo o impacto ambiental do descarte têxtil.*

## 3. Stakeholders
| Tipo | Interesse |
|------|-----------|
| Vendedores | Monetizar roupas não utilizadas |
| Compradores | Adquirir peças acessíveis |
| Doadores | Descartar roupas de forma sustentável |
| ONGs | Receber doações para comunidades |

## 4. Requisitos Principais
### Funcionais
- RF01: Cadastro de usuários com múltiplos perfis
- RF02: Sistema de publicação de anúncios com fotos
- RF03: Busca por filtros (tamanho, cor, marca)

### Não Funcionais
- RNF01: Tempo de resposta < 2 segundos
- RNF02: Compatibilidade com dispositivos móveis
- RNF03: Conformidade com LGPD
